## Vision

The fastest deck tool on the internet. Zero login, paste-and-go, instant card
previews, shareable URLs that just work, mobile-first, format legality, price
analysis, export to any client. Competes with Moxfield on speed and simplicity.

## Accepted Scope

### Core (Approach B baseline)
1. Delete dead app.jsx
2. Init git + .gitignore
3. Add /d/[encoded] route with SSR + OG meta tags
4. Fix Scryfall error handling (2 critical gaps: malformed JSON, null data)
5. Add tests (parser, scryfall, encoder)
6. Deploy to Vercel
7. Error boundary for crash recovery
8. Input length validation (10K char max)
9. Clipboard feedback toast on copy

### Accepted Expansions
1. **Deck Price Total** (S) — mainboard + sideboard + combined price from existing Scryfall data
2. **Format Legality Badges** (S) — which formats the deck is legal in, from existing Scryfall data
3. **Clipboard Paste Detection** (S) — auto-detect pasted decklists, prompt to view
4. **Export Formats** (S) — Copy as Arena / Copy as MTGO buttons
5. **Server-side Scryfall Cache** (S) — TTL cache for SSR /d/ route
6. **Image OCR** (M) — Tesseract.js client-side for decklist screenshots
7. **Mobile Touch Support** (S) — tap-to-expand card preview on touch devices
8. **Browser History Management** (S) — pushState for New Deck / Results navigation

## Not in Scope
- Deck Stats Panel (avg CMC, color distribution pie)
- PWA / Offline Support
- Deck persistence (database)
- User accounts / collections
- Deck comparison
- Meta analysis

## Critical Gaps to Fix
1. Scryfall malformed JSON response — SyntaxError crashes the app silently
2. Scryfall null data.data — TypeError crashes the app silently
3. No retry on 429 rate limit responses
4. ResolvedEntry interface duplicated in 2 files
